# The jQuery UI Audio widget enhances your HTML5 `<audio>`

## Demo
* <http://js.azatoth.net/audio/example.html>

## Features
* Adds full jQuery UI Themeroller support
* Pause, seek, rewind
* Volume control
* Scrub bar with time elapsed and remaining
* Unobstrusive and minimal UI
